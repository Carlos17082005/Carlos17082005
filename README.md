## 🚀 Proyectos Destacados

### ✈️ [Sistema de Gestión de Reservas y Pagos](https://github.com/Carlos17082005/Sistema_de_Gestion_de_Reservas_y_Pagos)
Plataforma web completa para gestionar, consultar y formalizar reservas de vuelos, integrando un flujo de pago simulado mediante la pasarela **Redsys** con firma `HMAC_SHA512_V2`.

- Autenticación de usuarios y control de sesiones con cookies
- Carrito de compras con verificación de stock de asientos en tiempo real
- Panel de historial de reservas con detalle de vuelos
- Interfaz responsiva con Bootstrap

### 🎓 [TestApp — Plataforma de Tests Educativos (TFG)](https://github.com/Carlos17082005/TFG_TestApp)
Aplicación web educativa Full Stack al estilo Duolingo que conecta profesores y alumnos dentro de módulos formativos. Trabajo de Fin de Grado desarrollado en equipo.

- Creación de preguntas de múltiples tipos: test, relacionar, selección múltiple
- Generación automática de tests únicos por alumno con filtros de dificultad
- Panel de seguimiento de resultados e historial por alumno
- Sistema de roles diferenciado: profesor / alumno

### 🎮 [Steam Python — Reseñas de Videojuegos](https://github.com/Carlos17082005/2526-PEP_PruebaEvaluacion2)
Plataforma web de reseñas de videojuegos inspirada en Steam, con tema oscuro y sistema de puntuación decimal. Desplegada en producción.

- Publicación de juegos con imagen, escritura de reseñas con puntuación (0.0–5.0)
- Sistema de autenticación y permisos por rol (anónimo / usuario / autor / staff)
- Soporte para múltiples backends de base de datos: SQLite y MySQL
- **Desplegada en:** [pythonanywhere.com](https://carlos17082005.pythonanywhere.com)
